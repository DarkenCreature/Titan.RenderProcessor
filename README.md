# Titan.RenderProcessor
a framework to render html dynamically in the frontend, based on JSON-dataobjects


``` html
<html>
  <head></head>
  <body>
    <test-component></test-component>
  </body>
</html>
```

Create Custom Component:

``` javascript
class TestComponent extends TitanComponent {
  constructor(){
    super();
  }
}
```
